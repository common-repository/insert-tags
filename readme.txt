Automatically insert the tag in your  site posts.

== Description ==

Automatically generates tags from title your posts  What then are inserted automatically in the postings on the site.

This plugin select the appropriate tags for better seo optimization.
It is a plugin suitable for bloggers and for communities with many users.
If you want to support us please  leave a review on wordpress.org
  or make a donation of 1 eur or more.





== Installation ==
1.  Upload plugin  in the directory  wp-content/plugins
2.  Access the admin panel and activate the plugin from the plugin section.
3. Accessing the settings insert tags.
